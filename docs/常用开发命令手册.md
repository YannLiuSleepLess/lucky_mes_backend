# 常用开发命令手册

## 一、数据库迁移

### 1. 生成迁移文件

```bash
cd src/AbpDemo.EntityFrameworkCore
dotnet ef migrations add <迁移名称>
```

示例：

```bash
dotnet ef migrations add Added_BasicData_Entities
```

### 2. 同步数据库（建表/更新表结构）

```bash
cd src/AbpDemo.EntityFrameworkCore
dotnet ef database update
```

### 3. 回滚到指定迁移

```bash
dotnet ef database update <目标迁移名称>
```

### 4. 删除最近一次迁移（未应用到数据库时）

```bash
dotnet ef migrations remove
```

### 5. 通过 DbMigrator 同步（含种子数据）

```bash
cd src/AbpDemo.DbMigrator
dotnet run
```

## 二、客户端代理生成

> 前置条件：先启动 Host 项目

### 1. 生成 Angular 前端代理

```bash
abp generate-proxy -t ng
```

### 2. 生成 C# 客户端代理

```bash
abp generate-proxy -t csharp
```

### 3. 指定模块和地址

```bash
abp generate-proxy -t ng -m AbpDemo --url https://localhost:44300
abp generate-proxy -t csharp -m AbpDemo --url https://localhost:44300
```

### 4. 移除已生成的代理

```bash
abp remove-proxy -t ng
abp remove-proxy -t csharp
```

## 三、项目运行

### 1. 启动后端 Host

```bash
cd src/AbpDemo.HttpApi.Host
dotnet run
```

### 2. 运行单元测试

```bash
dotnet test
```

## 四、解决方案级别操作

### 1. 编译整个解决方案

```bash
dotnet build
```

### 2. 还原 NuGet 包

```bash
dotnet restore
```

### 3. 清理编译输出

```bash
dotnet clean
```
